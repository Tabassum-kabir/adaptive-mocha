# Paper draft

`paper.tex` is the CHI 2027 submission scaffold. To compile:

```
pdflatex paper.tex
bibtex paper
pdflatex paper.tex
pdflatex paper.tex
```

The figures are generated by `python analysis/figures.py --tag mainstudy`
and land in `analysis/output/figures/`. Symlink or copy them into this
folder before compiling:

```
xcopy /Y ..\analysis\output\figures\*.pdf .
```

`fig_architecture.pdf` is hand-drawn; replace it with the final architecture
diagram before submission.

Every `\chiplaceholder{...}` in `paper.tex` is a literal hole that must be
filled with real-study numbers once the human data is in.

Use the official CHI 2027 LaTeX template once it is released; the prose
structure of this scaffold is template-agnostic and should slot into the
`acmart` `sigchi` class with no edits.
